# Generative UI Component Gym

A local React learning lab for controlled generative UI: known tool results select pre-authored,
schema-validated work cards. It uses synthetic execution data and does not require a model, credentials,
or external data source.

## Development

```sh
npm install
npm run dev
```

## Checks

```sh
npm run check
npm run test:browser
```

The initial shell establishes the deterministic toolchain. The subsequent tasks add trusted UI contracts,
stream simulation, approved work cards, validation, and the end-to-end demo.
